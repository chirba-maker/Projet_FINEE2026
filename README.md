# FINEE 2026 — Forum d'Innovation et d'Entrepreneuriat des Étudiants

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://finee-labe.gn)
[![Status](https://img.shields.io/badge/status-stable-green.svg)]()

Bienvenue sur le dépôt officiel du portail numérique **FINEE 2026**. Ce projet est une plateforme innovante conçue pour l'Université de Labé (Guinée), visant à digitaliser l'inscription, la soumission de projets et le suivi des participants au Forum d'Innovation et d'Entrepreneuriat des Étudiants.

---

## 🚀 Fonctionnalités Clés

### 💻 Dashboard Premium (SaaS Style)
Un tableau de bord moderne et intuitif permettant aux participants de :
- Suivre leur progression en temps réel.
- Accéder à leurs informations personnelles.
- Naviguer entre les différentes sections (Projets, Vote, Badge).

### 🏆 Système de Badge Digital
Génération automatique d'un badge officiel pour chaque participant :
- Design professionnel et moderne.
- Code QR unique pour vérification.
- Exportation facile (Format Image/PDF).

### 📂 Gestion de Projets
Interface dédiée à la soumission et à la gestion des projets innovants :
- Formulaire de soumission complet.
- Suivi de l'état d'avancement.
- Système de vote et classement intégré.

---

## 🛠️ Stack Technique

- **Frontend Core** : HTML5, CSS3, JavaScript (ES6+).
- **Styling** : [Tailwind CSS](https://tailwindcss.com/) pour une interface responsive et moderne.
- **Iconographie** : [Google Material Symbols](https://fonts.google.com/icons).
- **Stockage** : Persistence via l'API Web Storage (`localStorage`).

## ⚙️ Installation & Utilisation

1. **Clonage du projet**
   ```bash
   git clone https://github.com/votre-compte/finee-2026.git
   ```
2. **Lancement local**
   Ouvrez simplement le fichier `html/index.html` dans votre navigateur favori ou utilisez une extension type "Live Server" sur VS Code.
